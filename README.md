# Triangles Calc

מחשבון משולש ישר זווית אינטראקטיבי - אפליקציית PWA מותאמת לנייד.

## תכונות
- חישוב אוטומטי של צלעות וזוויות מתוך כל 2 נתונים מוגדרים (עם לפחות צלע אחת).
- שרטוט פרופורציונלי דינמי ב-SVG.
- הצגת שלבי פתרון מפורטים מתמטית עם רינדור נוסחאות ב-KaTeX.
- תמיכה ב-PWA (Progressive Web App) להורדה והתקנה כאפליקציה בטלפון הנייד או במחשב, כולל תמיכה במצב לא מקוון (Offline).
- כפתור ייעודי "הוסף למסך הבית" להתקנה קלה, אשר נעלם אוטומטית כשהאפליקציה מותקנת.
- הסברים והדרכה מובנים למשתמשי iOS Safari להתקנה ידנית.

## הוראות הרצה ופיתוח

### 1. התקנת תלויות
יש להריץ את הפקודה הבאה להתקנת Vite:
```bash
npm install
```

### 2. הרצת שרת פיתוח מקומי
יש להריץ את הפקודה הבאה לפתיחת שרת הפיתוח:
```bash
npm run dev
```

### 3. בנייה לגרסת ייצור (Production)
כדי לבנות את האתר לתיקיית `dist` מוכנה להעלאה ל-Cloudflare Pages או Github Pages:
```bash
npm run build
```
